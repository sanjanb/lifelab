Philosophy

This project is not a dashboard. It is a thinking surface.

If anything feels flashy, clever, or impressive — remove it.
If it feels boring but clear — you're doing it right.

Core Principles

  • No framework abstractions — Pure JavaScript
  • No hidden state — Everything is explicit
  • Deterministic rendering — Same input = same output
  • SVG for graphs — No canvas, no libraries
  • Data-first — Visuals serve the data
  • Works everywhere — Localhost = GitHub Pages

Data Model

Each day is a record tracking multiple domains on a 0-1 scale:

  • 0.0 — No engagement with this domain
  • 0.5 — Moderate engagement
  • 1.0 — Full engagement

The system is extensible — add new domains without breaking old data.

Technology Stack

  • Vite — Build tool (no framework)
  • Vanilla JavaScript — No abstractions
  • Pure SVG — No chart libraries
  • JSON files — No database
  • GitHub Pages — Static hosting

Privacy & Data

All data is stored locally in your browser using LocalStorage. Nothing is sent to external servers.

For backup and persistence:

  • Export your data as JSON or CSV
  • Save files to your local machine
  • Optionally commit to a private GitHub repository

Usage Guide

Daily Tracking

  1. Visit the Notebook page
  2. Use Quick Entry or edit individual days
  3. Rate each domain honestly (0.0 - 1.0)
  4. Add notes for reflection

Analysis

  1. Home page shows monthly trends
  2. Year page shows annual patterns
  3. Insights highlight patterns automatically

Data Management

  1. Export regularly for backup
  2. Import to restore or merge data
  3. Customize domains in Settings
