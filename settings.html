About LifeLab

Version 1.0.0
Stack Vite + Vanilla JS
Philosophy A thinking surface for self-tracking

Core Principles

  • No framework abstractions
  • No hidden state
  • Deterministic rendering
  • Data-first, visuals second

Data Storage

All data is stored locally in your browser and can be exported as JSON or CSV files. For persistent storage across devices, manually save your exported data and commit to GitHub.

Privacy

This application runs entirely in your browser. No data is sent to any server or third party.
